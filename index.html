Button Sizes

By default the base button class does not declare a font-size, so it is naturally set to 1em. Extensions to the.btn class are available for an additional three sizes:small,medium, andlarge.

If you'd like to extend or customize the size of the buttons, it's incredibly easy. Just apply a new font-size to the button and watch it shrink/grow.
